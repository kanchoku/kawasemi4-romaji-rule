# kawasemi4-romaji-rule
かわせみ4 の ローマ字ルール

このリポジトリには
[かわせみ4](https://www.monokakido.jp/ja/mac/kawasemi4/)
で「[Mozc で(だいたい) T-Code風入力](https://meech.hatenadiary.org/entry/20101222/1293037939)」と同じことを行うための
- ソースデータ roman.t-code.txt
- 変換プログラム oman.t-code.txt
- かわせみ4用ローマ字ルールのファイル roman.t-code.nrset
を収めています。
